## [직무 부트 캠프] SW 개발자 프로그램 개발 과정    
(2021.07.12 ~ 2021.08.09)

## 개발 환경
- OS : Windows10
- IDE : Intellij

## 개발 프레임워크
- jdk 11
- spring boot + web : spring과 web기능을 사용하기 위해서 사용 
- spring data jpa + h2 : 데이터 베이스 사용
- querydsl :  데이터 베이스 퀴리를 하기 위해서 사용
- lombok : 반복적 작업(getter,setter등) 처리시 사용
